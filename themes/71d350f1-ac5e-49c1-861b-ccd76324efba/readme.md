## Init Light
A rounded, boxy and light theme for Yasb. Just a minimal theme with focus on separation of utilities in the topbar and apps in the dock.

This theme uses a combination of Segoe UI, Segoe UI Fluent Icons and <a href="https://github.com/ryanoasis/nerd-fonts/tree/master/patched-fonts/JetBrainsMono"> JetBrainsMonoNL Nerd Font </a>
